About Taskwork

A user-friendly project management tool that utilizes the scrum framework, allowing users to track project tasks, review those in flight and boost team efficiency.

Guide to Taskwork

Step 1 • Create a new project. To do so, use the gray input field on the projects page. You have the ability to add a project title and description.

Step 2 • Click on the project you just created and proceed to the task screen. This is where you will track your to-do list, those 'in progress', 'in testing' and features that are complete.

Step 3 • Create a new task in the gray input field. Create as many tasks as needed to get started. You can always come back and add more.

Step 4 • Move the tasks through each phase using the green arrows to move forward, the blue arrows to go back or by pressing to delete.

Step 5 • When you are ready to submit tasks for review, use the 'mark for review' button to progress the project to 'under review' on the projects page.

Step 6 • Continue to work on your projects and tasks, use the mark as complete button when you have completed all the tasks.

Step 7 • If you no longer need the project, you can permanently delete it using the button at the end of the task page. Keep in mind this action is permanent and can not be un-done.
